# Borealis chess engine

Borealis is a chess engine that is currently under development. At the moment, it's in very early stages: i.e. learning the rules of chess.

This is a hobby project. I'm not looking to make money off it, but to increase my knowledge of C++ and of chess programming.

If you'd like to contribute, thanks for your interest! Frankly, this project isn't yet at a place where I'm looking for large-scale contributions, but that will change someday.

## Current status

- **Stage**: Very early alpha / prototyping
- **Language**: C++ 17
- **Platforms**: Building on Windows 11
